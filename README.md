# ZeroBox ✨

ZeroBox es una plataforma de almacenamiento en la nube inspirada en Google Drive, que integra funciones de inteligencia artificial para la edición avanzada de imágenes.  
Este proyecto representa tanto la versión inicial como la final, entregando una solución completa y lista para despliegue.

## 🚀 Características principales

- Gestión integral de archivos y carpetas (subida, descarga, renombrado, propiedades).
- Navegación clara mediante breadcrumbs y encabezados contextuales.
- Interfaz moderna, responsiva y consistente.
- Validaciones y mensajes de error en español.
- IA integrada para aplicar ediciones inteligentes a imágenes.
- Optimización de experiencia de usuario y semántica en labels y acciones.

## 🛠️ Tecnologías utilizadas

### Frontend

- **React 19** + **React DOM** – base de la aplicación.
- **React Router** – navegación entre carpetas y vistas.
- **TailwindCSS** + **tailwind-merge** – estilos modernos y responsivos.
- **Radix UI** – componentes accesibles y consistentes (Avatar, Dialog, Dropdown, Tooltip, Tabs, etc.).
- **Lucide React** – set de íconos vectoriales.
- **Next Themes** – soporte para modo oscuro/claro.

### Integraciones y lógica

- **Appwrite** – backend para autenticación y almacenamiento.
- **Axios** – cliente HTTP para integraciones.
- **ImageKit React** – optimización y manipulación de imágenes.
- **React Hook Form** + **Zod** – formularios con validación robusta.
- **Date-fns** – manejo de fechas con soporte de localización.
- **Sonner** – sistema de notificaciones.

### Desarrollo

- **TypeScript** – tipado estático.
- **Vite** – bundler rápido y moderno.
- **ESLint + Prettier** – linting y formateo.
- **typescript-eslint** – reglas avanzadas para TypeScript.
- **tw-animate-css** – animaciones con Tailwind.
